Submit Concession Boundaries

Concession Boundaries must be submitted in shapfile format and zipped in a folder. Please include the below Attributes:

Field Name Type Dropdown List Description
Commodity Text Commodity type (e.g. palm oil, soy, cocoa)
GroupName Text Group Name or RSPO Membership Name
CompanyName Text Company/Subsidary/Management Unit Name
PlantationName Text Plantation Name
SiteName Text Site Name
FacilityName Text Assoiciated processing facility name (e.g. name of mill, silo)
Type Text Permit type (e.g. HGU/land title/IUP)
PermitDate Date Month/Date/Year Permit Date
PermitNumber Text Permit Number
Area_ha Text Area in hectares
RSPOCertified Text True/False True means this plantation is RSPO certified (if applicable)
Certified_date Date Month/Date/Year RSPO certification assessment date (if applicable)
Certified_num Text RSPO certificate number (if applicable)

Submit Processing Facility Points

Processing facility points, in the form of coordinates, can be submitted in shapfile format in a zipped file, as a CSV, or as an Excel file. Please include the below Attributes:

Field Name Type Dropdown List Description
Commodity Text Commodity type (e.g. palm oil, soy, cocoa)
GroupName Text Group Name or RSPO Membership Name
CompanyName Text Company/Subsidary/Management Unit Name
SiteName Text Site Name
FacilityName Text Processing facility name (e.g. name of palm oil mill, soy silo)
Country Text Country processing facility is located in
PercentCover Short Integer Provide the percent of processing facility operations that the dataset represents. Put 100 if the dataset you have disclosed constitutes 100% of your company's facility operations.
RSPOCertified Text True/False True means this plantation is RSPO certified (if applicable)
Certified_date Date Month/Date/Year RSPO certification assessment date (if applicable)
Certified_num Text RSPO certificate number (if applicable)

Submit Smallholder Boundaries

Smallholder boundaries must be submitted in shapfile format and zipped in a folder. Please include the below Attributes:

Field Name Type Dropdown List Description
Commodity Text Commodity type (e.g. palm oil, soy, cocoa)
SmallholderType Text Smallholder type (e.g. scheme/associated, independent)
Number Long Integer Number of smallholder producers
Country Text Country smallholders are located in
Area_ha Float Area of smallholder production in hectares
FacilityName Text Assoiciated processing facility name (e.g. name of palm oil mill, soy silo)
PercentCover Short Integer Provide the percent of smallholder boundaries that the dataset represents. Put 100 if the dataset you have disclosed constitutes 100% of your company's smallholder boundaries.
RSPOCertified Text True/False True means smallholder farms are RSPO certified (if applicable)
Certified_date Date RSPO certification assessment date (if applicable)
Certified_num Text RSPO certificate number (if applicable)
